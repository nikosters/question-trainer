# Security Policy

## Как сообщить об уязвимости

Если вы нашли проблему безопасности, не публикуйте детали в открытом issue.

Используйте приватный security advisory в GitHub:

<https://github.com/nikosters/question-trainer/security/advisories/new>

Если advisory недоступен, создайте issue без технических деталей и укажите, что хотите передать информацию приватно.

## Что считается важным

- чтение или изменение файлов вне ожидаемого локального хранилища приложения
- выполнение произвольного кода через импорт JSON
- утечка содержимого пакетов, ответов или прогресса
- ошибки в release workflow, которые могут подменить APK
