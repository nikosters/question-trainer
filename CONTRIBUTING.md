# Как помочь проекту

Спасибо за интерес к проекту. Основной язык документации и интерфейса - русский.

## Перед началом

1. Создайте issue или обсудите изменение, если оно меняет поведение приложения.
2. Не добавляйте материалы экзаменов, задачники или тексты из других источников без явной лицензии на распространение.
3. Не меняйте пользовательские тексты интерфейса без отдельной причины: в приложении уже есть русская терминология.

## Локальная проверка

```bash
flutter pub get
dart format --output=none --set-exit-if-changed lib test
flutter analyze
flutter test
```

Если вы меняете логику импорта JSON, прогресс или работу теста, добавьте или обновите тесты.

## Pull request

- Делайте PR небольшим и сфокусированным.
- Опишите, что изменилось и как это проверить.
- Укажите, если изменение затрагивает формат JSON-пакетов или сохраненные данные пользователя.
- Убедитесь, что CI проходит.
