# Тренажёр заданий

Локальное Flutter-приложение для тренировки по пакетам заданий из JSON. Приложение импортирует JSON-файлы с вопросами, перемешивает вопросы и варианты ответов, сохраняет прогресс и помогает повторять ошибки.

Проект распространяется только через GitHub: исходный код находится в репозитории, готовые Android APK публикуются в GitHub Releases. Публикации в Google Play, App Store, F-Droid и pub.dev не планируются.

## Возможности

- импорт пакетов заданий из JSON
- локальное хранение пакетов и прогресса
- четыре варианта ответа на каждый вопрос
- рендеринг LaTeX-формул в тексте через `$...$`
- продолжение незавершенного теста
- повторная работа над ошибками

## Установка

1. Откройте страницу релизов:
   <https://github.com/nikosters/question-trainer/releases>
2. Скачайте APK из последнего релиза.
3. Установите APK на Android-устройство.

Android может попросить разрешить установку из браузера или файлового менеджера. Это нормально для APK, скачанных не из магазина приложений.

## Разработка

Требования:

- Flutter 3.35.7
- Dart 3.9.2
- JDK 17 для Android-сборок

```bash
git clone https://github.com/nikosters/question-trainer.git
cd question-trainer
flutter pub get
flutter run
```

Проверки перед pull request:

```bash
dart format --output=none --set-exit-if-changed lib test
flutter analyze
flutter test
```

## Формат пакета заданий

Пример пакета есть в [`example_package_ru.json`](example_package_ru.json). Подробное описание формата: [`docs/package-format.md`](docs/package-format.md).

Минимальный пример:

```json
{
  "questions": [
    {
      "id": "q1",
      "question": "Решите уравнение: $2x + 5 = 17$",
      "option_a": "$x = 5$",
      "option_b": "$x = 6$",
      "option_c": "$x = 7$",
      "option_d": "$x = 8$",
      "correct_option": "B",
      "explanation": "Переносим 5: $2x = 12$, значит $x = 6$.",
      "topic": "Линейные уравнения",
      "difficulty": "Базовый"
    }
  ]
}
```

## Приватность

Приложение работает локально. Оно не отправляет пакеты заданий, ответы, прогресс или аналитику на сервер. Подробнее: [`PRIVACY.md`](PRIVACY.md).
